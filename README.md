# Vortex Lotto Global Karma 2026

Ein non-profit Lotterie-Generator, der auf echtem Quantenzufall basiert und Karma-Punkte durch Spendenprojekte fördert.

## Features
- **Quanten-Zufall:** Zahlen werden live von der ANU Quantum Random Number Generator API bezogen.
- **Karma-System:** Über 100 integrierte Spendenprojekte (Bayern, Augsburg & Global).
- **Transparenz:** Vollständig Open Source und ohne Tracker.

## Datenquellen
- Lottozahlen: [ANU Quantum API](https://quantumnumbers.anu.edu.au/)
- Projekte: Dynamisch via `projekte.json`
